# UI 规格说明 — cc-monitor-bar

> 版本：1.0 · 日期：2026-04-02
> 依赖：[设计系统](./2026-04-02-design-system.md)

---

## 窗口架构

### 概览

| 组件 | 类型 | 尺寸 | 说明 |
|------|------|------|------|
| Popover | NSPopover | 320×480pt（动态） | 菜单栏弹出，快速查看 |
| 设置窗口 | NSWindow | ~520×400pt | 独立悬浮，支持实时预览 |

### 实时预览机制

设置窗口打开时，Popover 保持显示并实时响应设置变化。无需"应用"或"确定"按钮。

**实现方式**：
- 所有偏好设置通过 `@AppStorage` 持久化 + `@Published` 响应式绑定
- Popover 和设置窗口共享同一个 `AppPreferences` 实例
- 修改设置 → `@Published` 属性变化 → SwiftUI 自动刷新 Popover 内容
- 尺寸变化使用 `.animation(.easeInOut(duration: 0.25))` 平滑过渡

## 通用组件库

### StatCard — 统计卡片

**变体**：

| 变体 | 说明 |
|------|------|
| 数值型 | 标签 + 大数字 + 可选变化率 |
| 图标型 | 标签 + emoji/图标 + 大数字 + 辅助文字 |
| 进度型 | 标签 + 大数字 + 底部迷你进度条 |

**规格**：
- 圆角：`radius-md` (8pt)
- 内边距：14pt
- 边框：`Color(.separator).opacity(0.5)`
- 标签：`.caption` · `secondaryLabelColor`
- 数值：`.title3.bold` · `labelColor`
- 变化率/辅助：`.caption` · 状态色或 `tertiaryLabelColor`

### Badge — 徽章/标签

**用途**：模型标识、状态标记、数据分类

**规格**：
- 圆角：`radius-sm` (4pt)
- 内边距：3pt 10pt
- 背景：状态色 15% 透明度（如 `Color.systemBlue.opacity(0.15)`）
- 文字：`.caption` · 状态色 · 字重 medium
- 例：`Sonnet 4.6` → 蓝底蓝字，`活跃` → 绿底绿字

### ProgressBar — 进度条

**变体**：

| 变体 | 高度 | 用途 |
|------|------|------|
| 标准 | 4pt | 模型分布、用量占比 |
| 紧凑 | 3pt | 行内小型进度 |
| 多段 | 6pt | 多类别对比 |

**规格**：
- 圆角 = 高度 / 2（全圆角端点）
- 轨道色：`Color(.separator).opacity(0.5)`（自适应深色/浅色模式，避免硬编码白色值）
- 填充色：对应类别色（见设计系统模型/工具颜色映射）
- 标签：`.caption` 左对齐名称 + `.caption` 右对齐百分比
- 多段进度条无间距，各段紧密排列

### SessionRow — 会话行

**状态**：

| 状态 | 状态点颜色 | 行背景 |
|------|-----------|--------|
| 活跃 | `systemGreen` | `windowBackgroundColor.opacity(0.06)` |
| 历史 | `systemGray` | `windowBackgroundColor.opacity(0.03)` |
| 选中 | `systemGreen` | `windowBackgroundColor.opacity(0.08)` + 蓝色边框 |

**规格**：
- 状态指示点：8pt 圆形
- 行高：40-48pt
- 圆角：`radius-md` (8pt)
- 内边距：10pt 12pt
- 布局：`HStack` — 状态点 | 信息区(名称+副标题) | 数值区(数字+单位)
- 名称：`.subheadline` · `labelColor`
- 副标题：`.caption2` · `tertiaryLabelColor`
- 数值：`.subheadline.bold` · 强调色（活跃）或 `secondaryLabelColor`（历史）

### Divider — 分割线

- 颜色：`Color(.separator)`
- 上下间距：8pt
- 不使用全宽分割，两侧留 14pt 内边距

### FloatingNav — 浮动导航（看板视图专用）

**行为**：
- 滚动中：展开为完整标签（emoji + 文字），右对齐悬浮在内容区内部
- 静止 1.5s 后：收起为 4 个 8pt 小圆点
- 当前区块对应的圆点高亮为主题强调色

**规格**：
- 背景：`Color(.windowBackgroundColor).opacity(0.85)` + 毛玻璃
- 圆角：`radius-lg` (12pt)
- 边框：`Color.white.opacity(0.1)`
- 展开项内边距：5pt 8pt
- 展开项圆角：6pt
- 激活项背景：强调色 20% 透明度
- 动画参数：展开 200ms easeInOut / 收起 300ms easeInOut

### 其他组件

| 组件 | 规格 |
|------|------|
| Tab 切换器 (Segmented) | SwiftUI `Picker` + `.pickerStyle(.segmented)` · 圆角 6pt · 内边距 2pt |
| 微型柱状图 | 宽度 4pt / 间距 2pt / 圆角 2pt / 当前值高亮为主题色，其余为主题色 30-60% 透明度 |
| 状态指示点 | 尺寸 6-8pt / 纯色填充 / 无描边 |
| Tooltip (TipKit) | 圆角 10-12pt / 透明度 0.95 / 最大宽度 280pt / 标题 13pt / 正文 12pt |

**浮动导航 (FloatingNav)**：
- 悬浮在内容区右侧内部（非窗口外）
- 展开时最大宽度 60pt（emoji 16pt + 间距 6pt + 文字约 38pt），不挤压内容区——内容区右侧留有 68pt 的固定安全边距
- 收起时仅 20pt 宽（8pt 圆点 + 6pt 左右内边距），不影响内容阅读
- 滚动时展开为完整标签，静止 1.5s 后收起为小圆点
- 点击锚点可平滑滚动到对应区块
- 当前区块高亮（scroll spy）
- 参数见 FloatingNav 组件规格

## 设置独立窗口

### 窗口规格

| 参数 | 值 |
|------|-----|
| 类型 | `NSWindow`，风格 `.titled` + `.closable` + `.miniaturizable` |
| 尺寸 | ~520×400pt |
| 层级 | `.floating`（悬浮在普通窗口之上） |
| 激活策略 | 打开设置时不改变 `NSApp.activationPolicy`（保持 `.accessory`） |
| 窗口找回 | 设置窗口关闭后可通过菜单栏右键菜单 → "设置" 或 `⌘,` 快捷键重新打开。设置窗口持有强引用不会被释放，仅隐藏/显示 |
| 位置 | 首次打开时屏幕居中，之后记住用户最后拖拽位置 |

### 布局

macOS 标准设置窗口布局：左侧导航栏 + 右侧内容区。

**左侧导航**：
- 宽度：160pt
- 背景：略深于内容区
- 选中项：强调色 15% 背景 + 圆角 6pt
- 未选中：`secondaryLabelColor`
- 导航项：emoji + 文字，12pt

**右侧内容区**：
- 可滚动
- 每个设置项：标签 + 控件，行高 36-40pt
- 设置组之间用分割线分隔
- 标题 16pt bold，说明文字 12pt `secondaryLabelColor`

## 首次使用引导

使用 Apple TipKit（macOS 14+）实现轻量级渐进发现引导。

### 引导策略

| 阶段 | 触发时机 | 内容 |
|------|---------|------|
| 首次打开 Popover | 检测到首次启动 | Popover Tip 指向活跃会话区：「点击会话查看详情，按 Esc 返回」 |
| 首次打开设置 | 打开设置窗口时 | Popover Tip 指向主题选择：「切换主题可即时预览」 |
| 首次使用看板 | 切换到看板视图时 | Inline Tip：「滚动查看所有数据，右侧锚点可快速跳转」 |

### 引导参数

| 参数 | 值 |
|------|-----|
| 每个提示最大显示次数 | 2-3 次 |
| Popover Tip 透明度 | 0.95 |
| Popover Tip 圆角 | 10-12pt |
| 出现动画 | 200ms easeOut |
| 消失动画 | 150ms easeIn |
| 两个提示间最小间隔 | 3-5s |

### 实现

```swift
import TipKit

struct SessionDrillDownTip: Tip {
    var title: Text { "点击会话查看详情" }
    var message: Text? { Text("点击活跃会话可查看该会话的 Token、工具调用等数据。\n按 ⎋ Esc 返回今日总览。") }
    var options: [TipOption] { [MaxDisplayCount(3)] }
}

// 在视图中使用
.activeSessionList
    .popoverTip(SessionDrillDownTip(), arrowEdge: .top)
```
