## 产品定位

**cc-monitor-bar** 是一款 macOS 菜单栏原生应用，面向使用 Claude Code CLI 的开发者，提供本地使用监控能力。

核心价值：**不调用任何远程 API**，纯本地读取 `~/.claude/` 目录下的数据文件，零侵入地展示 Claude Code 的运行状态和历史用量。

## 用户场景

1. **实时监控**：开发者想了解当前有多少个 Claude Code 会话在运行，各自消耗了多少 Token
2. **用量分析**：回顾过去一周的 Token 消耗趋势、模型使用分布、工具调用频率
3. **成本感知**：通过 Token 统计推算使用成本，避免超额使用
4. **会话回顾**：查看历史会话的项目归属、持续时长、消息数量

## 核心功能

### 三种视图模式

| 视图 | 定位 | 目标用户 |
|------|------|---------|
| 极简视图 (MinimalView) | 一屏概览，今日统计 + 活跃会话 | 日常使用，快速了解 |
| 数据看板 (DashboardView) | 多维度分析，模型/会话/工具/趋势 | 深度分析用量 |
| 时间线 (TimelineView) | 按时间轴查看会话和事件 | 历史回顾 |

### 关键交互

- **会话钻取**：点击任意会话查看 Token 用量详情、模型分布
- **全局快捷键**：⌘1/2/3 切换视图、⌘, 打开设置、⌘R 刷新数据
- **浮动导航**：数据看板内的锚点快速跳转

### 设置

- **外观**：三种主题（原生自适应 / 毛玻璃通透 / Claude 暖调）
- **数据**：可配置刷新频率（3 秒 ~ 30 分钟）
- **快捷键**：查看和配置全局快捷键

## 非功能需求

- macOS 14.0+ 原生体验，菜单栏常驻
- 数据轮询不阻塞主线程
- 本地 SQLite 持久化，无需联网
